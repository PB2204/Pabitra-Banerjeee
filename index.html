Real Time Face Recognisition




I developed a real-time face recognition system using Python, OpenCV, and TensorFlow. It leverages OpenCV for face detection and feature extraction, while TensorFlow's deep neural network recognizes facial features and generates compact embeddings for matching. The system operates on a user-friendly web page.


By combining Python, OpenCV, and TensorFlow, I built a compact face recognition system. OpenCV handles face detection and feature extraction, while TensorFlow's deep neural network produces embeddings for matching. The system is accessible through a user-friendly web page.

Rotating Donut Animation




The project is a C program that generates a mesmerizing rotating donut animation using ASCII characters. It utilizes trigonometric functions to calculate coordinates and creates the illusion of rotation by continuously updating character positions. Shading effects enhance realism, and the animation rotates endlessly by incrementing angle parameters.


In summary, this C program combines ASCII art and programming to create a captivating donut animation. Trigonometry calculates coordinates, characters move to simulate rotation, and shading effects add depth. It demonstrates the artistic potential of mathematics and programming for visually engaging animations.
